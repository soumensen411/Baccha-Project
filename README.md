# Baccha Project 🧒

---

This is a **very baccha-type project** (a small and fun practice project).

It has a simple **login functionality**. When a user logs in, the profile picture magically changes into a **fancy Spiderman avatar** 🕷️🕸️ (●'◡'●).

Wahoo! 🎉

After logging in, the **user's name and email** also appear on the **profile card**.

That’s it.  
Simple. Small. Baccha. But still cool 😎.

---

## ✨ Features

- Simple login system  
- User name and email display  
- Fancy Spiderman profile avatar after login  
- Logout functionality  
- Built with **React + Context API**
